# Portfolio - Starter Scaffold

This is a minimal static portfolio scaffold using HTML, CSS, and JavaScript.

Files created:
- `index.html` — main single-page layout
- `styles/main.css` — base styles
- `scripts/main.js` — small interactive helpers (nav toggle, smooth scroll)
- `assets/images/` — placeholder SVG images
- `package.json` — convenience script to preview with `npx live-server`

Preview locally

Option A — open directly:

```powershell
start index.html
```

Option B — use `live-server` (recommended for auto-reload):

```powershell
npm start
```

If you don't have `live-server` installed, `npm start` uses `npx` to run it temporarily.
